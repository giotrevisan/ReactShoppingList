# React Shopping List

A single-page application built with React, TypeScript, and Vite that simulates an online shopping experience using mock data.

## Features

- Filter items by category  
- Select and manage favorite items with a constraint of up to 2 per category  
  - Automatically replaces the oldest selection when the limit is exceeded  
- Dedicated favorites page to review selected items  

## Technical Focus

This project emphasizes:
- State management and predictable data flow  
- Component-driven architecture  
- Enforcing business rules at the UI level  
- Handling user interactions with constrained logic  

## Tech Stack

- React + TypeScript  
- Vite  
- TailwindCSS  
- shadcn/ui  
- lucide-react  

## Setup

To run this project locally, ensure you have Node.js (version 20 or higher) installed.

```bash
npm install
npm run dev
```
